Колоквиум 8класс

1. Типы данных

Определение: Типы данных определяют, сколько памяти будет выделено под переменную, какой диапазон значений она может хранить и какие операции допустимы. Пример: целочисленные (int, long), вещественные (float, double), символьные (char). Правильный выбор типа помогает экономить память и избегать переполнений. 


// Пример разных типов
int signedInt = -100;
unsigned int unsignedInt = 100;
float f = 3.14f;
char c = 'A';

2. Системы счисления

Определение: Системы счисления — это способы записи чисел при различных основаниях (2, 8, 10, 16...). В C++ можно вывести одно и то же число в десятичном (dec), восьмеричном (oct) или шестнадцатеричном (hex) формате. 


// Пример перевода числа 255
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int num = 255;
    cout << "Десятичное: " << num << endl;
    cout << "Шестнадцатеричное: " << hex << num << endl;
    cout << "Восьмеричное: "      << oct << num << endl;
    return 0;
}

3. Случайные числа

Определение: Случайные числа в C++ генерируются функцией rand(). Чтобы получать разные последовательности при каждом запуске, используется srand(time(0)) (инициализация «seed»). Применяется в играх, тестировании, моделировании и т.д. 


#include <cstdlib>
#include <ctime>
#include <iostream>
using namespace std;

int main() {
    srand(time(0));
    int oneDigit   = rand() % 10;
    int twoDigit   = 10 + rand() % 90;
    int threeDigit = 100 + rand() % 900;
    cout << oneDigit << " " << twoDigit << " " << threeDigit << endl;
    return 0;
}

4. Деление

Определение: Целочисленное деление (оператор /) отбрасывает дробную часть. Оператор % возвращает остаток от деления. Например, 10 / 3 = 3, 10 % 3 = 1. 


int a = 10, b = 3;
cout << "Деление: " << a / b << endl;
cout << "Остаток: " << a % b << endl;

5. Условия

Определение: Условные операторы if, else, switch используются, чтобы выполнять код, когда условие истинно, или выбирать одну ветвь из нескольких. switch удобен при множестве вариантов одной переменной. 


if (x > 5) {
    cout << "Больше 5\n";
} else {
    cout << "Не больше 5\n";
}

switch (x) {
  case 5:  cout << "5";  break;
  case 10: cout << "10"; break;
  default: cout << "Другое";
}

6. Логика

Определение: Логические операции — && (И), || (ИЛИ), ! (НЕ). Возвращают булево значение (true/false). Приоритет: ! > && > ||. 


bool a = true;
bool b = false;
cout << (a && b) << endl; // 0
cout << (a || b) << endl; // 1
cout << (!a)     << endl; // 0

7. Цикл while

Определение: Цикл while запускает тело цикла, пока условие истинно. break прерывает цикл, continue пропускает оставшуюся часть итерации. 


int i = 0;
while (i < 5) {
    cout << i << " ";
    i++;
}

8. Цикл for

Определение: for чаще всего используют, когда заранее известно число итераций. Например, для суммирования ряда, вычисления факториала, перебора индексов массива. 


int sum = 0;
for (int i = 1; i <= 5; i++) {
    sum += i;
}
cout << "Сумма 1..5: " << sum;

9. Неизвестное количество

Определение: Когда нужно считать неизвестное число входных данных, обычно читают их в цикле до конца файла (EOF) или пока не встретится особое значение (например, 0). 


int sum = 0, val;
while (cin >> val) {
    sum += val;
}
cout << sum;

10. Массивы

Определение: Статический массив — это структура данных фиксированного размера, где элементы располагаются последовательно в памяти. Применяется, когда количество элементов заранее известно. 


int arr[5] = {1,2,3,4,5};
for (int i=0; i<5; i++) {
    cout << arr[i] << " ";
}

11. Обработка массивов

Определение: Обработка массива включает в себя операции, связанные с изменением порядка элементов (циклический сдвиг, сортировка, реверс), а также другие действия, меняющие структуру данных в массиве. 


#include <algorithm>

int arr[] = {1,2,3,4,5};
reverse(arr, arr + 5); // Переворот

12. Линейные алгоритмы

Определение: Линейные алгоритмы — одноразовый проход по массиву, чтобы найти минимальное/максимальное значение, посчитать сумму, проверить уникальные элементы и т.д. 


#include <set>
int arr[] = {1,3,3,7,5};
set<int> st(arr, arr+5);
cout << "Уникальных: " << st.size();

13. 2 указателя

Определение: Техника «два указателя» (start/end) используется для эффективного поиска подотрезков массива (например, с заданной суммой). Часто применяется к отсортированным данным. 


int arr[] = {1,2,3,4,5};
int start=0, end=0, sum=0, target=9;
while (end < 5) {
    sum += arr[end];
    while (sum > target && start <= end) {
        sum -= arr[start++];
    }
    if (sum == target) {
        // найден
    }
    end++;
}

14. Указатели и ссылки

Определение: Указатель хранит адрес переменной, а ссылка (reference) — «другое имя» той же переменной. Ссылка не может быть переназначена; указатель можно изменить или сделать nullptr. 


int x = 10;
int* ptr = &x; // указатель
int& ref = x;  // ссылка
*ptr = 20; // теперь x=20
ref = 30;  // теперь x=30

15. Массив & указатель

Определение: Имя массива (например, arr) указывает на адрес первого элемента (&arr[0]). Благодаря этому, к элементам можно обращаться как *(arr + i), что эквивалентно arr[i]. Это называется арифметикой указателей. 


int arr[] = {1,2,3,4,5};
int* p = arr; // p указывает на arr[0]
cout << *(p + 2); // 3

16. Векторы

Определение: std::vector — динамический массив в C++. Он автоматически расширяется или сокращается, позволяет добавлять и удалять элементы методами push_back(), pop_back() и т.д. 


#include <vector>
using namespace std;

vector<int> v;
v.push_back(10);
v.push_back(20);
cout << "Размер: " << v.size();

17. foreach

Определение: Range-based for (или «foreach») позволяет перебрать элементы массива или вектора без явного индекса. Пример: for (auto x : v). 


int arr[] = {1,2,3};
for (auto x : arr) {
    cout << x << " ";
}

18. Итераторы

Определение: Итераторы дают способ проходить по элементам контейнеров (vector, list, map и т.д.). С помощью begin(), end() (или rbegin(), rend() для обратного прохода) можно итерироваться, а методы insert(), erase() изменяют содержимое. 


vector<int> v = {1,2,3};
v.insert(v.begin()+1, 10); // вставка
v.erase(v.begin());        // удаление первого

// Обратный итератор
for (auto rit = v.rbegin(); rit != v.rend(); ++rit) {
    cout << *rit << " ";
}

19. ASCII

Определение: ASCII — кодировка, в которой символы (латинские буквы, цифры, знаки) имеют числовые коды от 0 до 127. В C++ можно приводить char к int, чтобы увидеть этот код. Функции isalpha, isdigit, tolower и т.д. упрощают работу с символами. 


#include <cctype>
#include <iostream>
using namespace std;

int main() {
    char c = 'A';
    cout << (int)c << endl;   // Код 'A' (65)
    cout << (char)tolower(c) << endl; // 'a'
    return 0;
}
